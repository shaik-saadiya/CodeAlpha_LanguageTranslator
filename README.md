# CodeAlpha Language Translation Tool (Java)

## Description
This project translates text from one language to another using an API.

## Features
- Supports English, Hindi, French
- Uses MyMemory Translation API
- Handles Unicode text properly

## How to Run
javac LanguageTranslator.java  
chcp 65001
java -Dfile.encoding=UTF-8 LanguageTranslator

## Example
Input: Hello  
Output: नमस्ते
